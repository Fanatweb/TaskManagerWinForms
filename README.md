# TaskManager — Управление задачами
![Скриншот](TaskManager.png)

## Описание
Программа для управления личными задачами. Позволяет добавлять, редактировать, удалять задачи, сортировать и фильтровать их.

## Возможности
- Добавление, редактирование, удаление задач
- Сортировка по ID, дате, приоритету, названию
- Фильтрация по статусу (Все / Не выполнены / Выполнены)
- Просроченные задачи выделяются красным цветом

## Как запустить
1. Откройте файл TaskManagerDudinRoman.sln в Visual Studio
2. Нажмите F5
